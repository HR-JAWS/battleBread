    public boardReducer(state: object , action: object ): * source 

    import boardReducer from 'battlebread/src/reducers/boardReducer.js'

    This function handles changes to the board state resulting from a state.dispatch(action)

    Params:

    NameTypeAttributeDescription
    state object

    state is an object representing each players' board. Example:
    state.board object
    state.board.turn number

    Turn counter
    state.board.p1Pieces number

    Number of remaining pieces for player 1
    state.board.p2Pieces number

    Number of remaining pieces for player 2
    state.board.p1 object

    Tile states for player 1
    state.board.p1[tileId] object

    Tile object, indexed by id string
    state.board.p1[tileId].id object

    Id string (ex: '1,1' or '6,5')
    state.board.p1[tileId].row number
    state.board.p1[tileId].col number
    state.board.p1[tileId].hasBread boolean
    state.board.p1[tileId].guessed boolean
    state.board.p1[tileId].color string

    'red' || 'green' || 'blue'
    state.board.p2 object

    Tile states for player 2
    state.board.p2[tileId] object

    Tile object, indexed by id string
    state.board.p2[tileId].id object

    Id string (ex: '1,1' or '6,5')
    state.board.p2[tileId].row number
    state.board.p2[tileId].col number
    state.board.p2[tileId].hasBread boolean
    state.board.p2[tileId].guessed boolean
    state.board.p2[tileId].color string

    'red' || 'green' || 'blue'
    action object

    an action dispatched via an action creator from ../actions.js

    Return:

    *

    Return Properties:

    NameTypeAttributeDescription
    type string

    'guess' || 'randomPieces' || 'setPiece
    payload object

    varies in shape for each action

    public guess(player: string, id: string) source 

    import {guess} from 'battlebread/src/actions.js'

    Performs a 'guess' action on a single tile. Depending on whether or not there is bread on that tile, different actions will occur.

    Params:

    NameTypeAttributeDescription
    player string

    either 'p1' or 'p2'. Represents TARGET player's board.
    id string

    the ID of the guessed tile, ex: '1,1' or '4,3'
